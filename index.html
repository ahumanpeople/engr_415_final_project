The Sense Hat

A 360° field-of-view obstacle detection system with haptic feedback for visually-impaired individuals. The device is built into a wide-brimmed hat using Time-of-Fight sensors, haptic motor, gyroscopic sensors, and a STM32 microcontroller.

Milton Tinoco · Kevin Huang · Jose Barrios
Explore the Project

Overview

Approximately 216 million people worldwide have a sort of visual impairment. To support individuals with visual impairments, this project proposes a wearable system that provides haptic feedback to help users perceive and navigate their surrounding environment, particularly in indoor settings. Current navigation methods, such as the use of a cane, can assist with obstacle detection but often require training and provide only limited environmental awareness.

The proposed system aims to improve upon these designs by providing a 360° sensing capability, allowing the user to receive feedback. Additionally, other research projects have explored integrating a similar system into common clothing items, such as work vests, but we have built upon this sensing system into a head-worn accessory to maintain a discreet and practical design.

Design Objectives

⚡

Reliability

Reliably detect nearby obstacles during indoor navigation, and scales in intensity as the distance between an object and the user, so the user is fully aware of its surroundings.

🎩

Discreet & Portable

All electronic components are concealed inside the hat's brim, with an additional enclosure to hold any larger components, such as the microcontroller and the battery. The assembled device remains lightweight and suitable for all-day wear.

💰

Affordability

The device should be designed with low-cost, off-the-shelf components, so that the price remains reasonably affordable for potential users.
